# EEG Label Marker

基于 Electron + WebGL2 的脑电波形标注工具，支持 EDF/BDF 格式文件的读取、显示与标注。

## 功能特性

### 文件支持
- 支持 EDF/EDF+/BDF 格式文件
- 支持同时打开多个文件
- 自动解析文件记录时间，显示绝对时间戳
- **文件关联**：安装后双击 `.edf` / `.bdf` 文件可直接打开
- **最近文件列表**：记录最近打开的文件，一键加载
- **拖拽打开**：将 EDF/BDF 文件拖入窗口即可加载

### 波形显示
- **GPU 加速渲染**：基于 WebGL2，流畅显示大量通道数据
- **自适应/固定高度模式**：自适应模式将所有通道压缩在一屏内；固定高度模式为每个通道分配固定像素高度，支持上下滚动浏览
- **时间轴缩放**：支持 10s ~ 10min 时间窗口
- **敏感度调节**：10μV ~ 3000μV 可调
- **绝对时间显示**：工具栏、时间轴、悬浮提示均显示文件记录的真实时间
- **通道标签宽度可调**：拖拽通道标签与波形之间的分隔线调整宽度，悬浮显示完整名称
- **全局时间轴概览条**：底部显示全段时间轴，标注以彩色色块呈现，支持点击跳转、拖拽平移、滚轮缩放，可切换显示/隐藏

### 信号处理
- **陷波滤波**：50Hz / 60Hz 工频干扰滤除
- **高通滤波**：0.1Hz ~ 10Hz
- **低通滤波**：15Hz ~ 100Hz
- **基线校正**：去均值 / 去中位数，实时生效
- **双极导联**：自动识别连续编号通道，生成链式双极导联（如 LG1-LG2, LG2-LG3, ...）

### 通道操作
- **通道选择面板**：搜索、全选、全不选
- **左键点击**选中通道（高亮加粗显示）
- **左键拖拽**框选多个通道
- **右键拖拽**取消选中
- **坏道管理**：自动检测坏道，支持手动标记/取消，独立导出/导入坏道文件，可筛选只显示坏道

### 标注功能
- **中键点击**自动进入标注模式并选择时间点
- **自定义标注类型**：可添加/删除/重命名标注类型及颜色
- **标注备注**：每条标注可添加文本备注
- 标注区间在波形上高亮显示（半透明背景 + 边缘色带）
- 标注列表可视化管理，支持单条删除和清空
- **点击标注列表条目跳转**：点击标注自动跳转到对应时间位置
- **标注统计汇总**：标注面板底部显示各类型数量和总时长
- **导出**标注为 TXT/CSV 文件（附统计摘要）
- **导入**已有标注文件
- **自动保存**：定时自动保存标注，防止数据丢失

### 频谱分析
- **实时 FFT 功率谱分析**：对选中通道弹出 FFT 窗口，对数/线性坐标可切换
- **时频分析（Spectrogram）**：选定通道显示时频图，支持多种色图（Viridis/Jet/Hot/Inferno），线性/对数刻度切换

### 交互操作

| 操作 | 自适应模式 | 固定高度模式 |
|------|-----------|-------------|
| 滚轮 | 滚动时间轴 | 滚动时间轴 |
| 左键拖拽 | 滚动时间轴 | 上下滚动通道 + 左右滚动时间轴 |
| 左键点击 | 选中通道 | 选中通道 |
| 左键框选 | 框选多个通道 | 框选多个通道 |
| 右键拖拽 | 取消选中 | 取消选中 |
| 中键点击 | 标注时间点 | 标注时间点 |
| Ctrl+滚轮 | 缩放时间轴 | 缩放时间轴 |

## 安装与运行

### 环境要求
- Node.js >= 18
- npm

### 开发模式

```bash
# 安装依赖
npm install

# 启动应用
npm start
```

### 打包为安装程序

```bash
# 打包（Windows 下生成 NSIS 安装包）
npm run build
```

打包产物位于 `dist/` 目录：
- `EEG波形标注工具 Setup 1.0.0.exe` — NSIS 安装包，支持选择安装位置
- `win-unpacked/EEG波形标注工具.exe` — 解压版

安装后自动关联 `.edf` / `.bdf` 文件，双击即可打开。

> 国内网络环境下，打包前需设置 Electron 镜像：
> ```powershell
> $env:ELECTRON_MIRROR="https://npmmirror.com/mirrors/electron/"
> $env:ELECTRON_BUILDER_BINARIES_MIRROR="https://npmmirror.com/mirrors/electron-builder-binaries/"
> ```

## 标注导出格式

导出为纯文本文件，每行一条标注，字段以制表符分隔：

```
通道名    起始时间    结束时间    标签    备注
```

时间格式为绝对时间 `HH:MM:SS.xxx`（基于 EDF 文件记录起始时间）。

文件末尾附带统计摘要：

```
# --- 统计摘要 ---
# 发作期    3  12.5s
# LVFA      2   8.0s
# 合计      5  20.5s
```

示例：
```
EEG LG 01-Ref    22:33:12.500    22:33:15.300    ictal    发作起始
EEG LG 02-Ref    22:33:12.500    22:33:15.300    ictal
```

导入时同时支持新格式（通道、绝对时间、标签、备注）和旧格式（秒数、通道、标签）。
